# 🤖 Basic Python Chatbot

A simple **rule-based chatbot** built using Python.

## 📌 Features

-   Rule-based responses
-   Continuous conversation until `bye`
-   Beginner friendly

## 📂 Project Structure

``` text
Basic-Chatbot/
├── chatbot.py
└── README.md
```

## 🚀 Run

``` bash
python chatbot.py
```

## 💬 Supported Commands

  Input                Response
  -------------------- -----------------------------
  hello                Hi! Nice to meet you.
  hi                   Hello!
  how are you          I'm fine, thanks!
  what is your name?   My name is Basic Chatbot.
  who created you      I was created using Python.
  thank you            You're welcome!
  bye                  Goodbye!

## 🧠 Concepts Used

-   Functions

-   if-elif-else
-   input/output
