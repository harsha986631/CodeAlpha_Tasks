# 🎮 Hangman Game

A simple and interactive **Python Hangman Game** where players guess a
hidden word one letter at a time.

## ✨ Features

-   Random word selection
-   Multiple categories
-   Input validation
-   Limited attempts
-   Easy-to-use command-line interface

## 🛠️ Built With

-   Python 3

## 📂 Project Structure

``` text
Hangman/
├── hangman.py
└── README.md
```

## 🚀 Installation

``` bash
git clone https://github.com/your-username/hangman-game.git
cd hangman-game
python hangman.py
```

## 🎮 Gameplay

Guess the hidden word by entering one letter at a time. Each incorrect
guess reduces your remaining attempts. Guess the word before you run out
of chances!
